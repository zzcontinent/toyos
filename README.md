# toyos 
## quick start
```
cd src
make
make qemu-nox
make qemukill
```
## content
```
0 tools: os开发环境工具链、bios、bootloader、kernel调试方式
1 asm: 汇编语言(x86)
2 启动流程
3 pmm物理内存管理
4 vmm虚拟内存管理
5 内核线程管理
6 用户进程管理
7 调度器
8 同步互斥
9 文件系统
10 display + game
```
